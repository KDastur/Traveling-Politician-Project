# Traveling Politician Project 

### Description
The objective is to find the most efficient route for a politician to travel to every state capital once,
starting in Iowa, and ending in Washington, DC.
(This program calculates the route and distance when traveling between all the capitols originally not including Alaska and Hawaii, then it calculates the distance if you were to take a round trip to alaska and hawaii from the closest capitol)
